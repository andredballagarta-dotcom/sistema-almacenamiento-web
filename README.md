## Descripcion 
 
Sistema web para la gestion de almacenamiento de equipos de computo, desarrollado con Java Servlets, JSP y MySQL. 
 
--- 
 
## Tecnologias Utilizadas 
 
- **Backend:** Java 17, Servlets, JSP 
- **Frontend:** HTML5, CSS3 
- **Base de Datos:** MySQL 8.0 
- **Servidor:** Apache Tomcat 10.1 
- **Build Tool:** Maven 3.9
